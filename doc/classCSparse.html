Detailed Description

CSparse represents sparse matrix:
All data is stored as map of pair (int, int) and float value where pair of ints is coordinates of the non-zero value
Every non-zero value takes 12 bytes + some overhead for map and pair
+: Stores data much more efficient for matrices with many non-zero values than 'regular' 2d matrix;
Sparse matrix does not store zero values
-: Has logarithmic complexity for both getting and setting values while 'regular' dense matrix has constant
